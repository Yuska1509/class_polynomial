# class_polynomial
python 2020 ДЗ 2
Реализация класса Polynomial (многочленов с целыми коэффициентами) в питоне с помощью стандартной библиотеки Python. А также двух производных классов RealPolynomial (многочленов нечетной степени с вещественными коэффициентами) и QuadraticPolynomial (многочленов степени не выше 2 с целыми коэффициентами) со специфическими методами.
